# Sales Performance Dashboard (Excel)

## Project Overview
This project is an interactive Sales Performance Dashboard built in Microsoft Excel to analyze sales data across regions, products, and salespersons.

## Features
- KPI Cards: Total Sales, Total Quantity Sold, Average Sale Value
- Interactive Slicers (Region, Product, Salesperson)
- Pivot Tables & Pivot Charts
- Clean and professional dashboard design

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Formulas (SUM, AVERAGE)
- Dashboard Design & Data Analysis

## Use Case
This dashboard helps management track sales performance and make data-driven decisions.
